# olympic-games-analytics-dashboard--powerbi
A comprehensive Olympic Games Analytics Dashboard built using Power BI to analyze over a century of Olympic history. The project explores athlete performance country-wise medal distribution, sport-wise trends. It transforms raw sports data into interactive visual insights to understand dominance, participation trends and evolution of global sports.
🏅 Olympic Games Analytics Dashboard
📊 Project Overview

This project presents a comprehensive analysis of the Olympic Games dataset from 1896 to 2016, covering both Summer and Winter Olympics. The dataset contains 271,616 rows and includes athletes, sports, countries, medals, gender, and event-wise information.

The goal is to convert raw Olympic data into meaningful insights using Power BI dashboards.

🎯 Objectives
Analyze Olympic performance trends from 1896–2016
Study gender-wise participation (male vs female athletes)
Evaluate country-wise and sport-wise medal distribution
Identify top-performing athletes and nations
Provide interactive insights using dashboards and slicers
🛠️ Tools Used
Power BI
Power Query (Data Cleaning & Transformation)
Excel Dataset

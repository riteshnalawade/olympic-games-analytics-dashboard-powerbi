# 🏅 Olympic Games Analytics Dashboard (1896–2016)

## 📊 Project Overview

A comprehensive **Olympic Games Analytics Dashboard** built using Power BI to analyze over a century of Olympic history (1896–2016).

The project explores:
- Athlete performance  
- Country-wise medal distribution  
- Sport-wise trends  
- Gender participation patterns  

It transforms raw sports data into **interactive visual insights** to understand dominance, participation trends, and evolution of global sports.

---

## 🎯 Objectives

- Analyze Olympic performance trends (1896–2016)  
- Study gender-wise participation (male vs female athletes)  
- Evaluate country-wise and sport-wise medal distribution  
- Identify top-performing athletes and nations  
- Build interactive dashboards using slicers and filters  

---

## 🛠️ Tools Used

- Power BI  
- Power Query (Data Cleaning & Transformation)  
- Excel Dataset
